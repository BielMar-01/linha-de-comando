### MOVIMENTAR (BASH)

Move para a pasta sites

```shell
$ cd sites
```

Move para a pasta anterior

```shell
$ cd ..
```

Move para a pasta sites dentro de Desktop

```shell
$ cd ~/Desktop/sites
```

Move para o diretório principal do sistema / home

```shell
$ cd ~
```

Limpa a tela

```shell
$ clear
```
